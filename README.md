# erl_list_bench

Benchmark for testing the performance of different list implementations.

## Prerequisites

The app uses `rebar v3+` package manager.

### Get Dependencies

`$ make get-deps`

## Compile

### Binary

`$ make compile`

### Run

Note that `run` automatically calls `get-deps`.

`$ make run`

### Shell

Note that `shell` automatically calls `get-deps`.

`$ make shell`

## Dependencies

* eministat (`https://github.com/jlouis/eministat`)
